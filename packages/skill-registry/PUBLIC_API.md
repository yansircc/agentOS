# @agent-os/skill-registry Public API

Status: 0.2.x public-experimental. Skill registration is install-time identity
over core tool contracts; it is not a runtime ledger/event/projection surface.

## Public exports

None.

## Experimental exports

- `.:RegisteredSkill`
- `.:RegisterSkillResult`
- `.:SkillManifest`
- `.:SkillRegistryIssue`
- `.:SkillToolManifest`
- `.:UnregisterSkillResult`
- `.:registerSkill`
- `.:unregisterSkill`

## Internal-only exports

Any package file or symbol not listed above.

# @agent-os/skill-registry Public API

Status: public-experimental. Skill registration is install-time identity over
core tool contracts; it is not a runtime ledger/event/projection surface.

## Frozen exports

None.

## Experimental exports

- `.:RegisteredSkill`
- `.:RegisterSkillResult`
- `.:SkillManifest`
- `.:SkillRegistryIssue`
- `.:SkillToolManifest`
- `.:UnregisterSkillResult`
- `.:registerSkill`
- `.:unregisterSkill`

## Internal-only exports

Any package file or symbol not listed above.

Deferred Exclusions (spec-35)

The following components are explicitly omitted from this substrate operator console dashboard, as per operator scope guidelines.

  • no approve / publish / discard action — operator decisions are app saga, not ops-api
  • no app-noun columns (change, lead, site, schema, conversation)
  • no cost rollup — needs explicit cost projection; quota is not cost
  • no cross-scope fan-in view — spec-30 watchlist until O(10) connection pressure
  • no business charts (deals / inquiries / customers)
  • no mutable state surface — every value here is project(events)
  • opaque scopes (artifact-blob/…, sandbox/…) are listed but runTrace/runStatus/quota/... return 501 not_introspectable
